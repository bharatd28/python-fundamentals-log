# My Python Learning Journey

Welcome to my personal coding repository. 

This project is where I am documenting my progress as I learn Python from the ground up. Instead of just reading documentation, I am saving my daily practice scripts, algorithmic puzzle solutions, and breakthroughs here to track my growth over time.

---

## What is Inside

I have broken down my files as I master each core programming concept:

* **The Basics:** Working with strings, escape sequences, and standard output formatting.
* **Control Flow:** Writing conditional logic with if-else structures and executing loops.
* **Data Collections:** Understanding how Python manages data groups using Lists, Tuples, Sets, and Dictionaries.
* **Problem Solving:** My practice vault for solving algorithmic challenges from platforms like LeetCode.
